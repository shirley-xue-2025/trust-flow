## Anti-Jira / ServiceNow

> "We're not building another IT ticket queue or ServiceNow approval form. TrustFlow replaces email chains with a structured agent boardroom — and compiles the outcome into code the gateway enforces."

> "No workflow engine theater — agents negotiate; humans sign once; the edge enforces deterministically."

---

## Hybrid sovereign deploy

> "Layer A gateway runs in your VPC — PII scan, routing, deny codes stay on your side. Layer B boardroom runs on Qwen Cloud where multi-agent negotiation belongs. Best of sovereign edge and sponsor-aligned agent compute."

> "Hackathon demo is docker-compose; production story is hybrid — gateway colocated with employee traffic, boardroom SaaS with your DPA."

## PII honest script

> "Email is masked — the model sees `[EMAIL_MASKED]` and the request can continue. IBAN is hard-blocked — `PII_BLOCK`, no exfil. We don't restore PII after masking; audit logs fingerprints, not raw prompts."

> "This is regex policy for the demo — production would add proper NER, but enforcement stays deterministic Layer A code, not LLM judgment."

**Do NOT say:** Presidio, round-trip PII restore, "enterprise-grade NER" (unless scoped honestly).
