The Dual-Layer Architecture

We separate Deterministic Execution (Programmatic Layer) from Agentic Decision-Making (Intelligent Layer). This ensures legal safety is absolute, while policy creation remains flexible.

Layer A: The Edge Gateway

A high-performance programmatic proxy that executes local rules. It handles PII masking and HTTP routing with zero hallucination.

Layer B: The Agent Boardroom

A multi-agent network (Legal, IT, Runner) that negotiates parameters and dynamically compiles them into Layer A's ruleset.
