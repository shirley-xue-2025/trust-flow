# Competitor adaptation prototypes (R6)

Standalone HTML demos showing UX patterns adapted from **TrendAI** and **Naaia** competitor research into TrustFlow.

**Open locally:** start at [`index.html`](index.html) (double-click or `python3 -m http.server` from this folder).

| Prototype | Inspired by | TrustFlow component |
|-----------|-------------|---------------------|
| [`01_request_qualification.html`](01_request_qualification.html) | Naaia Assess + onboarding | Request UI → pre-boardroom qualification |
| [`02_compliance_command_center.html`](02_compliance_command_center.html) | Naaia Core + Registry + Event Tracker | Admin UI compliance score, action plans, auditor view |
| [`03_gateway_control_plane.html`](03_gateway_control_plane.html) | TrendAI AI Guard + AI-SPM | Layer A gateway rules, monitor/enforce, registry health |
| [`04_bridge_end_to_end.html`](04_bridge_end_to_end.html) | Both + TrustFlow wedge | Full flow: request → boardroom → compile → enforce |

**Research:** [`docs/research/06_competitor_inspiration_for_trustflow.md`](../../docs/research/06_competitor_inspiration_for_trustflow.md)

**Scenario data:** Claude Code / payments team ([`scenario_001`](../../docs/research/roleplay/scenario_001_payments_claude.md)).
